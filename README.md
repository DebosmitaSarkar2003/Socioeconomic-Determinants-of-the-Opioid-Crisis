
# Socioeconomic Determinants of the Opioid Crisis

## 📊 Project Overview
The opioid epidemic is a complex public health challenge influenced by a variety of social and economic factors. This project performs a statistical deep-dive into these determinants, using data science to identify high-risk environments and underlying trends that contribute to the crisis.

## 🚀 Key Features
- **Statistical Modeling in R:** Implementation of regression analyses to quantify the impact of socioeconomic variables.
- **Data Visualization:** Geographic and demographic charts highlighting the hardest-hit regions and populations.
- **Multidimensional Analysis:** Examining the intersection of poverty, unemployment, and healthcare infrastructure with opioid misuse rates.
- **Policy Recommendations:** Data-backed insights aimed at informing targeted public health interventions.

## 📁 Repository Structure
- `Opioid_Analysis_FINAL.R`: The complete R script containing data processing, statistical tests, and visualization code.
- `Final Project Presentation.pptx`: A visual summary of the research questions, methodology, and key findings.
- `Final Group Project Rough Draft.pdf`: Detailed documentation and literature review supporting the analysis.
- `README.md`: Project overview and documentation.

## 🛠️ Tech Stack
- **Language:** R
- **Libraries:** Tidyverse, ggplot2, dplyr, broom (for statistical modeling)
- **Documentation:** Microsoft Word, PowerPoint
